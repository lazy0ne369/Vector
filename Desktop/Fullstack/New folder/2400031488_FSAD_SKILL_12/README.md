# Student Management System

Full-stack CRUD app using Spring Boot and React.

## Features
- Add, View, Update, Delete students
- React (Axios) ↔ Spring Boot REST API
- No page reload (state updates)

## Tech Stack
- Backend: Spring Boot, JPA, MySQL
- Frontend: React, Axios

## API
- POST /students
- GET /students
- PUT /students/{id}
- DELETE /students/{id}

## Run

### Backend
Run Spring Boot app (port 8080)

### Frontend
cd frontend
npm install
npm start
